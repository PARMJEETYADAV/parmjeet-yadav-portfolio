# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/PARMJEETYADAV/parmjeet-yadav-portfolio.git
```

Move into the project directory

```bash
cd parmjeet-yadav-portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:3000
```

---

# 📦 Build for Production

```bash
npm run build
```

Run the production server

```bash
npm start
```
